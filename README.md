# vue-dice-roller
 3D dice roller made in VueJS using CSS 3D transforms.

 See it in action [here](https://stevennicklin.github.io/vue-dice-roller).

 ## Installation

Install using yarn

```bash
yarn install
```

## Usage

### Development
Run locally in develop mode
```bash
yarn dev
```

### Static page
Build static page
```bash
yarn build
```

Serve static page
```bash
yarn preview
```
